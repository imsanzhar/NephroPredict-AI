NephroPredict AI — созылмалы бүйрек ауруын (СБА) ерте кезеңде анықтауға арналған заманауи интеллектуалды жүйе. Жоба негізінде кестелік деректерге арналған трансформер архитектурасы (FT-Transformer) жатыр.

🚀 Жобаның мақсаты
Бұл жоба медициналық зертханалық талдаулардың 10 негізгі көрсеткішіне сүйене отырып, пациенттің СБА-ға шалдығу ықтималдығын есептейді. Басты назар — ауруды жіберіп алмау (High Recall) қабілетіне аударылған.

🏗 Технологиялық стек

Тіл: Python 3.12

Framework: PyTorch (Терең оқыту)

Web Interface: Streamlit

Деректерді өңдеу: Scikit-learn, Pandas, NumPy, SMOTETomek (Hybrid Sampling)

Архитектура: FT-Transformer (Feature Tokenizer + Transformer Encoder)

📊 Модельдің сипаттамасы
Модель жай ғана нейрондық желі емес, ол Self-Attention механизмін қолданады. Бұл дегеніміз:
Feature Tokenization: Әрбір клиникалық көрсеткіш (мысалы, Креатинин немесе GFR) жеке векторлық токенге айналдырылады.
Multi-Head Attention: Модель көрсеткіштер арасындағы күрделі байланыстарды талдайды.
Clinical Priority: Шекті мән (Threshold) 0.3 деп орнатылған, бұл ауруды ерте анықтау деңгейін (Sensitivity) арттырады.

Модель метрикалары (5-Fold Cross-Validation):
Accuracy: 89.36%
Recall (Sensitivity): 93.02% 🌟
Precision: 95.24%
F1-Score: 94.12%
ROC-AUC: 0.8102

🛠 Қолданылатын көрсеткіштер
Модель диагноз қою үшін келесі 10 параметрді талап етеді:
Қандағы глюкоза (FBS) және HbA1c
Сарысудағы креатинин және BUN
Гломерулярлы сүзу жылдамдығы (GFR)
Зәрдегі белок (Protein in Urine)
Натрий (Sodium)
Гемоглобин (Hemoglobin)
Жалпы және HDL холестерин
